# React + tailwindcss

This is the movie app that work using react and tailwind css for best ui.

## Installation

### 1,Create react project

```sh
create npm vite@latest movie-app
then choose  .react
            .javascript
```

### 2,Go to project folder

```sh
cd movie-app
```

### 3,install dependencies.

```sh
npm install
```

### 4,Install Tailwindcss

```sh
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
```

### 5,Run project

```sh
npm run dev
```
